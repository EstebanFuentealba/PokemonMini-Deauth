ESP32-S3 XIAO

Conecta el XIAO por USB. Chrome o Edge pueden instalar directamente el firmware UART/WiFi.

  1. Desconecta el RP2040 del UART durante la instalación.
  2. Pulsa instalar y elige el puerto del XIAO.
  3. Confirma la instalación segura.
Web Serial requiere Chrome o Edge de escritorio. Descargar aplicación .bin
03

RP2040 / PM2040

El UF2 incluye la ROM y el puente mailbox/PIO-UART para GPIO27 (TX) y GPIO28 (RX), basado en el bus de cartucho de PM2040.

  1. Mantén BOOT presionado al conectar el RP2040.
  2. Espera que aparezca la unidad RPI-RP2.
  3. Selecciona esa carpeta para copiar el UF2.
Descargar firmware .uf2

Modo laboratorio. El firmware ESP32 escanea redes y puede monitorizar pasivamente tramas deauth/disassociation del AP seleccionado. No transmite deauth ni inyecta paquetes. El RP2040 usa UART PIO a 115200 baud en GPIO27/28.
